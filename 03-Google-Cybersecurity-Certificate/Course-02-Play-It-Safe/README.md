# 🛡️ Play It Safe: Manage Security Risks

Notes and study materials for **Course 2** of the **Google Cybersecurity Professional Certificate**.

## Topics Covered

- Security Risk Assessment
- CIA Triad
- Security Controls
- NIST Cybersecurity Framework (CSF)
- NIST Risk Management Framework (RMF)
- SIEM & SOAR
- Security Audits
- Threats, Risks, and Vulnerabilities

## Repository Structure

- `Notes.md` – Personal study notes
- `Glossary.md` – Cybersecurity terms and definitions
- `Controls.md` – Types of security controls
- `NIST-CSF.md` – NIST Cybersecurity Framework
- `NIST-RMF.md` – NIST Risk Management Framework
- `Security-Risk-Assessment.md` – Risk assessment concepts

## Goal

Build a solid understanding of cybersecurity risk management and security frameworks while preparing for future security analyst roles.
